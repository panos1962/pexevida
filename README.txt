Το project "pexevida" αποτελεί μια απόπειρα υλοποίησης στο διαδίκτυο
των παιχνιδιών "μπουρλότο" και "βίδα", όπως αυτά παίζονται στην Ελλάδα.
Το project εδράζεται σε μια σχεσιακή database στην οποία κρατούνται
τα στοιχεία των παικτών, οι σχέσεις μεταξύ των παικτών, η αλληλογραφία τους
και οι παρτίδες που παίζονται.
Ο κορμός των προγραμμάτων αποτελείται από ένα σύνολο κλάσεων και μεθόδων
που αναπαριστούν όλα τα φυσικά αντικείμενα της εφαρμογής (παίκτες, καφενεία,
παρτίδες, τραπουλόχαρτα, χαρτωσιές κλπ). Ως κύρια γλώσσα προγραμματισμού
χρησιμοποιείται η javascript, ενώ στον server χρησιμοποιείται ο Node.js
προκειμένου το interface να είναι κοινό σε server και clients.

filajs API
----------
Το API "filajs" θα πρέπει να υπάρχει εγκατεστημένο στον server στο directory
"client/filajs", ή να υφίσταται σχετικό symbolic link "client/filajs" που να
δείχνει στο API.
