Στο παρόν directory έχουμε εικόνες, εικονίδια και background patterns
που αφορούν στο project "pexevida":

	vida16.ico		Εικονίδιο βίδας μεγέθους 16x16 pixels.

	vida32.ico		Εικονίδιο βίδας μεγέθους 32x32 pixels.

	vida32.png		Εικονίδιο βίδας μεγέθους 32x32 pixels.

	vida64.png		Εικονίδιο βίδας μεγέθους 64x64 pixels.

	vida128.png		Εικονίδιο βίδας μεγέθους 128x128 pixels.

	vida256.png		Εικονίδιο βίδας μεγέθους 256x256 pixels.

	vidaGmail.png		Background pattern για το Google mail.

	vidaTwitter.png		Background pattern για το Twitter.

	vidaFacebook.png	Φωτογραφία προφίλ για την ομάδα "pexevida"
				του Facebook.
